# Android Application Development

This repository groups course assignments by chapter and language version, plus the end-of-semester project.

## Structure

- `Assignment/Chapter 1/Student Grade Calculator/Dart Version`
- `Assignment/Chapter 1/Student Grade Calculator/Kotlin Version`
- `Assignment/Chapter 2/Dart Version`
- `Assignment/Chapter 2/Kotlin Version`
- `End of Semester Project/CamVote`

## Notes

- Chapter work is organized under the `Assignment` folder exactly as requested.
- Chapter 1 imports the existing Student Grade Calculator projects with their original git history preserved.
- Chapter 2 includes the Kotlin exercises repository with preserved history and a new Dart version that runs from the console.
- `CamVote` is imported into the end-of-semester folder with its existing main-branch history preserved.
