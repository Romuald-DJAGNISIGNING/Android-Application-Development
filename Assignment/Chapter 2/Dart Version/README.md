# Chapter 2 Dart Version

Dart console exercises and milestone demos for Chapter 2.

## Contents

- `functions_exercises.dart`: higher-order functions, filter/map/fold style collection work.
- `class_exercises.dart`: abstract classes, interfaces, sealed-style states, and polymorphism.

- `milestone3_romualdsigning.dart`: object-oriented grading model with abstract components and interface-based policy.
